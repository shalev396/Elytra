# Express.js Backend Template for E-Commerce

This is a placeholder for the Express.js backend implementation for E-commerce projects.

## Features

- Fast, unopinionated, minimalist web framework for Node.js
- Robust set of features for web and mobile applications
- HTTP utility methods and middleware support
- High performance routing

## Getting Started

```bash
cd backend
npm install
npm start
```
